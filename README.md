# 1️⃣ Classificador de Nível de Herói

## 📋 Descrição

Nesta atividade, desenvolvemos um classificador em JavaScript que determina o **nível de um herói** com base na quantidade de experiência (XP) acumulada.

## 🎯 O que é feito

Duas variáveis armazenam o **nome** e a **quantidade de XP** do herói. Com base no XP informado, uma estrutura de decisão identifica o nível correspondente, que pode ser: Ferro, Bronze, Prata, Ouro, Platina, Ascendente, Imortal ou Radiante.

## 🛠️ Conceitos utilizados

- **Variáveis** — para armazenar o nome e o XP do herói
- **Operadores** — para comparar os valores de XP com as faixas de nível
- **Estruturas de decisão** — `if/else` ou `switch` para classificar o nível do herói

## 📤 Saída esperada

```
O Herói de nome {nome} está no nível {nivel}
```
